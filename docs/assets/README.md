Placeholder directory for MVP demo and portfolio screenshots.

**Do not commit large binary files without optimization.** Replace each placeholder below with a real PNG when capturing the demo.

## Required screenshots

| File | Capture | Suggested content |
|------|---------|-------------------|
| `dashboard-home.png` | Full Dashboard | All 4 zones + header with business name and goal |
| `priority-card.png` | Today's Priority | Signals, Why this action, Expected impact, badges, feedback buttons |
| `learning-card.png` | CampusFin Learning | Latest recommendation, feedback, learned text, recent actions, progress |
| `feedback.png` | Feedback modal | Helpfulness modal after tapping ✓ 已执行 |

---

## How to capture

1. Run `npm run dev` and log into demo account
2. Complete today's Daily Check-in
3. Use browser devtools device mode (390×844) or desktop (1280×800)
4. Save PNGs to this folder with exact filenames above
5. Reference in README.md Demo Screenshots section
